About VEXLearn

VEXLearn is a template designed for learning VEX V5 programming in C++. It provides a structured codebase with PID-based movement functions, a display system, and a clean architecture to help new programmers get started quickly. It is intended for tank-drive V5 robots using the VEX VS Code extension.

This guide was written from personal experience over two years of VEX competition, covering both the technical aspects of the template and the practical realities of being a coder on a VEX team.
